    Projetos:

  • Bot de Notificação: Neste projeto, enfrentamos desafios no processamento de dados e no consumo excessivo de banda satelital em um sistema crítico que consumia dados de um satélite por meio de um microserviço. Para ajudar na resolução desses problemas, implementei um bot de notificação para o Microsoft Teams, utilizando a ferramenta de automação de workflows N8N. O bot fornecia atualizações em tempo real sobre problemas no microserviço, permitindo uma resposta rápida e eficiente da equipe de operações e desenvolvimento.

    As notificações também ajudaram a identificar padrões e tendências nos problemas, permitindo análises mais profundas e a implementação de soluções preventivas. Como resultado, conseguimos melhorar a eficiência operacional, reduzir o tempo de processamento do microserviço e otimizamos o uso da banda satelital.

  • Dashboard de Monitoramento: Criei um dashboard no grafana para monitorar, em tempo real, problemas em equipamentos que se comunicam com nosso microsserviço satelital, permitindo respostas rápidas da equipe de suporte e desenvolvimento.

    A solução aprimora a eficiência do suporte e oferece uma visão estratégica dos problemas críticos.

  • Integração com Sistema Araucária: Participei de um projeto voltado ao desenvolvimento de APIs como extensão do nosso sistema, com o objetivo de exportar dados em formato JSON para o sistema de nosso cliente chamado Araucária. Minha atuação incluiu a implementação da autenticação e autorização disponibilizando usuário e senha exclusivo para essa integração, garantindo segurança na troca de informações.

    Também criei mecanismos para configurar limites nos registros retornados e desenvolvi endpoints, como o de snapshots, que fornece as últimas informações registradas de cada equipamento, incluindo posição geográfica, estado operacional, hora do último evento e status atual. Além disso, desenvolvi o endpoint de telemetria, que retorna dados completos do trajeto e telemetria dos equipamentos desde a última sincronização.

    Essa solução otimizou a integração entre os sistemas, garantindo eficiência e confiabilidade para monitoramento e análise operacional dos equipamentos.

  • Processo de Gestão de Vulnerabilidades: Neste projeto desenvolvi e implementei um processo estruturado para gestão de vulnerabilidades, integrando boas práticas de desenvolvimento seguro. Conduzi o primeiro ciclo de testes de intrusão (pentest) da organização, identificando e priorizando vulnerabilidades críticas nos sistemas.

    Também elaborei documentações e relatórios técnicos detalhados, incluindo planos de mitigação e recomendações para fortalecer a segurança. Além disso, atuei diretamente na correção das falhas mais críticas, garantindo a proteção de ativos sensíveis e a conformidade com padrões de segurança.

  • Tecnologias: C#, .NET 6, PostgreSQL, Entity Framework, API REST, N8N e Vue.js.

    Isso não só eliminou a tarefa manual de atualização, mas também aumentou a produtividade da equipe de desenvolvimento, permitindo que se concentrassem em novas funcionalidades e resolução de problemas em produção.

  • Integração com E-commerce: Com o objetivo de integrar o e-commerce de nosso cliente ao nosso ERP, criei uma solução que facilita todas as etapas do processo de gerenciamento de pedidos. Agora, nosso cliente pode, gerar vendas, emitir notas fiscais e despachar encomendas tudo de maneira intuitiva e simplificada.

    Isso resultou em um aumento significativo na produtividade da entrega de pedidos, causando um aumento aproximado de 70% no faturamento do cliente.

  • Integração com Porto Seguro: Desenvolvi uma integração com o sistema de averbação de carga do porto seguro, substituindo o processo manual e propenso a erros por uma solução automatizada.

    Isso eliminou problemas de acompanhamento e atrasos associados à comunicação via email, proporcionando uma transmissão segura e precisa de dados através de nossa plataforma, poupando tempo valioso do cliente e acelerando o fluxo de trabalho.

  • Controle de Produção de Pedidos de Restaurantes: No cenário dinâmico e acelerado dos restaurantes, onde a eficiência é essencial, desenvolvi uma interface kanban para otimizar a gestão dos pedidos.

    Essa ferramenta visual e organizada melhorou o fluxo de produção, aumentando a produtividade na entrega dos pedidos, garantindo a satisfação do cliente.

  • Automação para Dropshipping: Desenvolvi uma solução de automação para um fornecedor de dropshipping, desenvolvendo rotinas de integrações de cadastro de produtos, processamento de pedidos, rastreamento de envios e atualizações de preços e estoques em tempo real.

    Como resultado a solução melhorou a eficiência operacional e fortaleceu parcerias com vendedores.

  • Importação de dados: Criei uma solução eficiente de importação de dados que importa informações de planilhas para nosso ERP. Identifiquei dados necessários para adicionar em nossa estrutura, criei rotinas para extração dos dados e implementei uma interface intuitiva que permitia o upload das planilhas.

    Após implementação, houve uma redução significativa no tempo de importação e melhoria na precisão dos registros. A solução proporcionou uma vantagem competitiva, agilizando o processo de captação de novos clientes.

  • Integração com App de Delivery: Desenvolvemos uma solução de integração para aplicativos de delivery, facilitando o gerenciamento de pedidos, rastreamento de entregas e processos gerenciais de restaurantes. Implementei integrações para melhorar a eficiência operacional e a experiência do cliente.

  • Integração com Correios: Para atender a necessidade de simplificar e agilizar o processo de envio de produtos, criei uma solução integrada aos correios para automatizar o processo de envio. Neste projeto desenvolvi rotinas de integração para postagem de objetos, geração de etiquetas e inclusão de códigos de rastreamento nos pedidos de forma automática.

    Após implementação, houve uma redução significativa no tempo de preparo e envio de pedidos, melhorando a eficiência operacional.

  • Tecnologias: C#, .NET Framework, SQL Server, API REST, HTML, CSS, JavaScript (jQuery).
